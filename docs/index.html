GitHub Pages 无法实现真正的自动桥接，需要：

  • Jitsi + Jigasi（SIP 网关）
  • Zoom Meeting Bot SDK
  • 媒体服务器处理音视频流

❓ 常见问题

Q: 为什么 Zoom 不能嵌入网页？

A: Zoom 设置了安全策略，禁止在 iframe 中嵌入。这是为了防止钓鱼攻击。


Q: 如何实现真正的自动桥接？

A: 需要使用服务器运行桥接程序，GitHub Pages 只能提供静态网页。


Q: 有没有完全免费的自动桥接方案？

A: 可以尝试使用 Oracle Cloud 的永久免费服务器 + Jigasi 方案。
